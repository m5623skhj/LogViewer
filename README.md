# LogViewer

## 개요

`LogViewer`는 여러 로그 파일을 확인하기 용이하도록 지원하는 WPF 기반 로그 뷰어입니다.

### 주요 기능
1. **다중 로그 파일 동시 로드** (.log, .txt)
2. **시간 기반 필터링** (시작/종료 시간 지정)
3. **다중 키워드 검색** (AND/OR 지원)
4. **로그 레벨별 색상 구분** (Info, Warning, Error, Debug)
5. **핀 기능**

---

## 주요 UI 정보

### 로그 레벨별 색상
- **Info**: 밝은 초록색
- **Warning**: 카키색
- **Error**: 연한 산호색
- **Debug**: 하늘색

### 검색 하이라이트
- 검색된 라인: 노란 배경
- 핀 설정된 라인: 회색 배경

---

### 사용 방법
1. **파일 열기** 버튼으로 로그 파일 선택 (다중 선택 가능)
2. 검색어 입력 후 **검색** 버튼 클릭 (공백으로 키워드 구분)
3. 시간 필터: `yyyy-MM-dd HH:mm:ss` 형식으로 입력 후 적용
4. 중요 로그는 📌 버튼으로 핀 설정
5. **핀 항목만 보기** 버튼으로 핀 설정된 로그만 필터링

---

## 로그 파싱

### 지원 형식
- JSON 형식 로그 (여러 줄에 걸친 JSON 자동 병합)
- 타임스탬프 형식: `yyyy-MM-dd HH:mm:ss`
- 로그 레벨 키워드: ERROR, WARN, DEBUG (대소문자 무관)
